PSoC/Arduino Calculator

Description

The final project for our Microcontroller System Design class was to design a microcontroller system which utilized PSoC 1 and the Arduino Uno to create a basic calculator. Using the SPI bus, both microcontrollers were able to communicate back and forth. We also tested the logic and communication between the microcontrollers using Saleae 16 Logic.
